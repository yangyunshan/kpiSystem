中国地质大学(武汉)地理与信息工程学院地理系教育教学绩效计算办法（试行）

为激励系教师勤于教育教学、潜心教育教学、热爱教育教学，本着营造老师在系教育教学方面 “都参与”、“有贡献”的氛围，针对学院发展和专业建设的需要，加强并创新系管理体制，特制定以下绩效办法，记分标准见附件一。

相关说明如下：

绩效记分原则

本办法将计入老师们为系教育教学与日常运行管理“用了心思”、“花了时间”、“耗了精力”、“出了成果”的所有工作事件（一流专业建设、教书育人、指导与辅导学生学习/科技活动/成长成才、系务活动等）。

第二条：绩效记分项类型

1) 按工作项计分 主要为系教学、学生管理、人才培养等工作，涉及教学排名、教材出版、教学项目申报、教学比赛、院/系活动等具体项。 2) 按工作量计分 主要为一些更体现工作量的工作项，以“次数”或“天数”为单位记录。如：课程组研讨、听课、学生宿舍走访、指导优秀学生获奖、帮扶困难学生次数等。 3) 按金专贡献值计分 主要为国家双一流专业申报涉及的项，要求为省部级以上级别，涉及项包括：教学成果、名师与教学团队、课程与教材、专业建设、实验和教学实践平台、教改项目、指导学生教育教学奖励。

第三条：绩效评定委员会

组长：吕建军、胡楚丽 成员：季军良、祁昆仑、曾杰、杨乃、陈旭、林晓、周超

第四条：相关规定

1. 所有成果不重复统计。 2. 所有数据由个人或相关负责人报送，不得弄虚作假，否则取消数据报送人其当年所有绩效。 3. 有违“以德立身、以德施教、以德立学、以德育德”的教师，取消其当年的所有绩效。 4. 个人绩效得分用作当年系年终福利分配和评优的依据。

本办法相关条文由地理系负责解释，未尽事宜由本绩效评定委员会会议商议。
